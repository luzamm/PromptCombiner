# 关键词保存在/prompts里面了
只要是在这个文件夹里的并且后缀为plist的文件都会被读入,各位可以自己添加关键词.关键词目前还不完善,我也只录了少部分上去,希望各位多多帮忙,也能快点完善这个简陋的软件
# plist文件格式 
分类^中文名称^prompt^介绍
e.g
身体姿态^站立一字马^standing split^站立一字马是xxxxxxx
